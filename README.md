# Sound Controlled Smart Light System

## 📌 Project Overview
The **Sound Controlled Smart Light System** is an IoT-based automation project that controls a light using sound commands such as a clap. The system detects sound through a sound sensor and automatically switches the light ON or OFF using a microcontroller. This project demonstrates a simple and efficient approach to home automation.

---

## ✨ Features
- Clap-based light control
- Automatic ON/OFF switching
- Low power consumption
- Easy to build and use
- Cost-effective home automation solution
- Suitable for smart home applications

---

## 🛠️ Components Used
- Arduino Uno
- Sound Sensor Module
- Relay Module
- LED/Bulb
- Breadboard
- Jumper Wires
- Power Supply

---

## ⚙️ Working Principle
1. The sound sensor continuously monitors surrounding sounds.
2. When a clap is detected, the sensor sends a signal to the Arduino.
3. The Arduino processes the signal.
4. The relay switches the connected light.
5. A second clap changes the light back to its previous state.

## 🚀 Installation
1. Clone the repository.
```bash
git clone https://github.com/your-username/Sound-Controlled-Smart-Light-System.git
```

2. Open the Arduino IDE.

3. Connect the Arduino Uno to your computer.

4. Open the `.ino` file.

5. Select the correct board and COM port.

6. Upload the code to the Arduino.

---

## ▶️ Usage
- Power the Arduino circuit.
- Clap once to turn the light ON.
- Clap again to turn the light OFF.
- Repeat as needed.
